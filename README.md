# DAZN Shortcuts

Adds useful keyboard shortcuts to the DAZN.com player using the WebExtensions API.

## Shortcuts
| Key | Description |
|---|---|
| `k` | Toggle play/pause |
| `f` | Toggle fullscreen |
| `m` | Mute video |
| `l` | Go to LIVE stream (if behind) |
| `arrow left` | Skip backward |
| `arrow right` | Skip forward |
